================================================================================
                          DATABASE SCHEMA
================================================================================

PROFESSIONAL MODEL (Professional.js):
--------------------------------------
{
  name: String (required),              // Professional's full name
  number: String (required),            // Contact phone number
  location: String (required),          // Service area/location
  category: String (required),          // Main category (e.g., "House Needs")
  service: String (required),           // Specific service (e.g., "Electrician")
  rating: Number (default: 0),          // Average rating (0-5)
  experience: String (required),        // Years of experience
  verified: Boolean (default: true),    // Verification status
  reviews: [ReviewSchema],              // Array of customer reviews
  timestamps: true                      // createdAt, updatedAt
}

REVIEW SCHEMA (Embedded in Professional):
------------------------------------------
{
  user: String (required),              // Reviewer's name
  rating: Number (required, 1-5),       // Star rating
  comment: String (required),           // Review text
  createdAt: Date (default: Date.now)   // Review timestamp
}

================================================================================
                          API ENDPOINTS
================================================================================

1. GET /api/local-help/professionals
   ----------------------------------
   Description: Fetch professionals with optional filters
   Access: Protected (requires JWT token)
   Query Parameters:
     - location (optional): Filter by location
     - category (optional): Filter by category
     - service (optional): Filter by specific service
   
   Response:
   {
     "success": true,
     "count": 12,
     "data": [
       {
         "_id": "...",
         "name": "Amit Sharma",
         "number": "+91 92345 67890",
         "location": "smriti nagar",
         "category": "House Needs",
         "service": "Electrician",
         "rating": 4.5,
         "experience": "5 years",
         "verified": true,
         "reviews": [...]
       },
       ...
     ]
   }

2. POST /api/local-help/review/:id
   --------------------------------
   Description: Add a review for a professional
   Access: Protected (requires JWT token)
   URL Parameter: id (Professional's MongoDB _id)
   Body:
   {
     "rating": 5,
     "comment": "Excellent service!"
   }
   
   Response:
   {
     "success": true,
     "data": {
       // Updated professional object with new review
     }
   }

3. POST /api/local-help/seed
   --------------------------
   Description: Seed database with demo professionals (Development only)
   Access: Public (should be protected in production)
   Body: None
   
   Response:
   {
     "success": true,
     "message": "Seeded successfully",
     "count": 450
   }
